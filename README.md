# Beautiful Java Programming

## Requirements

```
1. Java 11

2. Maven 3.x.x
``````

## Setup

**1. Git Clone**

```
git clone https://github.com/lsefiane/beautiful-java-programming-sample.git
``````

**2. Maven Build**

```
cd beautiful-java-programming-sample/

mvn clean install
``````
